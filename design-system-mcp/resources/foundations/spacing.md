# Spacing

## Scale

| Token | Value | Usage |
|---|---|---|
| `space-0` | 0 | Reset |
| `space-1` | 4px | Tight: icon gaps, badge padding |
| `space-2` | 8px | Compact: button padding, small gaps |
| `space-3` | 12px | Default: form field spacing |
| `space-4` | 16px | Standard: card padding, section gaps |
| `space-6` | 24px | Comfortable: between sections |
| `space-8` | 32px | Spacious: page padding |
| `space-12` | 48px | Large: between major sections |
| `space-16` | 64px | Extra large: page top/bottom |

## Component Spacing

| Context | Spacing |
|---|---|
| Between form fields | `space-3` (12px) |
| Card padding | `space-4` (16px) |
| Between cards in grid | `space-4` (16px) |
| Between page sections | `space-8` (32px) |
| Page margin (mobile) | `space-4` (16px) |
| Page margin (desktop) | `space-8` (32px) |

## Rule

Use the spacing scale consistently. Never use arbitrary values like `13px` or `37px`.
